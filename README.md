# Alurakut

Um projeto feito na Imersão React da Alura.

### 🤔 Como Usar

```
   - Clone esse repositório:
   $ git clone https://github.com/MuriloBarreto/alurakut.git

   - Entre no diretório:
   $ cd alurakut

   - Instale as dependências:
   $ npm install

   - Inicie o projeto: 
   $ npm run start
   
  ```

  * [Link do projeto](https://alurakut-murilobarreto.vercel.app/)
